RICE VARIETY PHOTOS DIRECTORY
==============================

This directory should contain photos of your rice varieties.

Current Rice Varieties (see scripts/main.js):
- ponni.svg - Ponni (replace with ponni.jpg when you have a photo)
- bpt.svg - BPT (replace with bpt.jpg when you have a photo)
- ir20-deluxe-ponni.svg - IR20 Deluxe Ponni (replace with ir20-deluxe-ponni.jpg when you have a photo)
- idly-rice.svg - Idly Rice (replace with idly-rice.jpg when you have a photo)

Placeholder SVG images are included. To use your own photos: add .jpg/.png files with the same base names and update the image paths in scripts/main.js from .svg to .jpg

Image Recommendations:
- Format: JPG, PNG, or WebP
- Size: 800x800px (square) for best display
- File size: Optimize for web (under 300KB each recommended)

To add or modify rice varieties, edit the riceVarieties array in scripts/main.js
